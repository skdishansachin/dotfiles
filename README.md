# Config

Personal configuration and dotfiles that managed with GNU Stow.

## Usage

Clone the repo into `~/config/` (or any directory), then run:

### Dotfiles

```sh
stow bash
stow nvim
stow starship
```
Each package corresponds to a folder containing the files to be symlinked.

**Official Documentation for GNU Stow** - https://www.gnu.org/software/stow/

## Setting up a new machine?

### Intall

You can just run the `./install.sh` script to install the other softwares and font I use.

### Font

I was using system Monospace font which alies to "DejaVu Sans Mono".
